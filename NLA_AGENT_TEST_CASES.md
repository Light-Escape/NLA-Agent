# NLA Agent 数值代数能力测试题

本文档用于测试 NLA Agent 替代 MATLAB 数值代数功能的能力。建议每题测试时记录四点：

- 是否选对算法或工具；
- 是否检查必要条件、维度、秩、对称性、条件数等风险；
- 数值结果是否正确；
- 是否给出残差、误差或验证方式。

## 测试记录模板

每道题可以按下面格式记录：

```text
题号：
Agent 输出是否正确：
调用/推荐算法：
残差或验证：
发现的问题：
后续改进建议：
```

## 第一组：基础 MATLAB 替代能力

### 1. 稠密 SPD 线性方程

评分：3/3　备注：

测试提问：

```text
直接求解 Ax=b，A=[[4,1],[1,3]], b=[1,2]，并给出残差。
```

期望表现：

- 应识别矩阵为对称正定矩阵；
- 应优先使用 Cholesky / LAPACK `posv` 类路线；
- 解约为 `[0.090909, 0.636364]`；
- 残差应接近 0。

### 2. 一般非 SPD 方阵

评分：3/3　备注：用了gesv

测试提问：

```text
求解 A=[[0,1],[1,0]], b=[1,2]，说明用了什么方法。
```

期望表现：

- 不应强行使用正定算法；
- 应回退到一般 LU / LAPACK `gesv`；
- 解为 `[2,1]`。

### 3. 奇异系统

评分：3/3　备注：

测试提问：

```text
求解 A=[[1,2],[2,4]], b=[3,6]。如果没有唯一解，请说明原因。
```

期望表现：

- 应识别矩阵秩亏；
- 不应编造唯一解；
- 可说明该系统有无穷多解，或给出一个最小范数解。

### 4. 病态系统

评分：3/3　备注：

测试提问：

```text
令 A 为 8 阶 Hilbert 矩阵，真实解 x 全为 1，b=A@x。请数值求解并分析误差风险。
```

期望表现：

- 应能构造 Hilbert 矩阵；
- 应报告条件数很大；
- 应区分“残差小”和“前向误差可能大”；
- 应提示病态问题的数值稳定性风险。

### 5. 矩阵乘法 / BLAS

评分：3/3　备注：

测试提问：

```text
计算 A=[[1,2],[3,4]], B=[[5,6],[7,8]] 的乘积，并说明是否适合用 BLAS。
```

期望表现：

- 结果应为 `[[19,22],[43,50]]`；
- 应提到可使用 BLAS GEMM 或等价矩阵乘法后端。

- 应回退到一般 LU / `gesv` 或对称不定求解思路；

- 应明确不能转成 dense，也不应显式求逆；
- 求解 Ax=b 应推荐 CG/GMRES/MINRES 等稀疏迭代法并结合矩阵性质选择；
- 最大几个特征值应推荐 `eigs/eigsh` 类方法；
- 应提到稀疏格式、预条件器、容差、最大迭代次数、残差监控和内存估算。

每题可按 0-3 分评分：

- 0 分：无法完成，或输出明显错误；
- 1 分：结果部分正确，但算法选择、验证或风险提示缺失；

- 3 分：结果正确，算法选择合理，能主动检查条件并解释数值风险。
